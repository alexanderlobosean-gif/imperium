Debug Variáveis de Ambiente

Abra o console do navegador (F12) e veja os logs para identificar qual .env está sendo usado.

Passos:

  1. Abra: http://localhost:5173
  2. Abra o console do navegador (F12)
  3. Procure por logs como:
    • import.meta.env.VITE_SUPABASE_URL
    • import.meta.env.VITE_SUPABASE_ANON_KEY
    • Invalid API key
    • Auth state changed
  4. Se vir "Invalid API key", o .env atual tem chaves erradas
  5. Se vir "undefined", as variáveis não estão sendo carregadas

Arquivos .env existentes:

  • .env - Chaves antigas/inválidas
  • .env.local - Chaves corretas (se você atualizou)
  • .env.example - Template de exemplo

Solução:

Se as chaves do .env estiverem erradas, copie as chaves corretas do dashboard Supabase e substitua no arquivo.
